Participation

Participation is one of the easiest parts of the SEP class. All you need to do is try. Raise your hand when the teacher asks a question even if you aren't sure of your answer. You will still get points even if you are wrong.

Homework

The SEP class does not give that much homework in the first place. Thats why homework is especially important. If you miss one homework that could bring your grade down significantly.

Quizzes,Tests,Portfolio

Quizzes and Tests are obvously a big part of your grade in any class and thats no different for the SEP class. The difference is that its really easy to study for. The SEP website has slides that you can use to go over the material. Its a very useful tool to use to study.

SEP9

Scratch:

Scratch is the first Unit of SEP. It is a program where you can make games by using blocks to make what appears on the screen. This is an intro to coding concepts and its an easy first start.

Robotics:

Robotics is the second Unit of SEP. This unit was uses the EV3 software to make the robot do certain things. You can make the robot run an obstacle course by inputting certain commands on the computer which go to the robot.

Python:

Python is the third Unit of SEP. Python is a programming language that is a good beginners course for coding languages.

SEP10

Processing:

Processing is the first unit SEP10. Its an open source programming language. Its more advanced then Python but still helps you with the fundamentals of coding. An example of a basic line of code would be:

void setup() { println("Hello world."); }

Makey Makey:

Makey Makey is the second Unit of SEP10. In this unit you learn about electricity and how it flows between circuits. No coding is needed because you are using an Arduino which is a prototype kit that lets you interact with things in different ways.

Arduino:

Arduino is the third Unit of SEP10. In this unit you focus more on the hardware of Arduino than in the previous unit. The Arduino itself is an open source hardware which means that people can add on to improve the device. You use code to make the arduino do different things.

Data:

Data is the last Unit of SEP10. In this unit you learn about what data is exactly. With that knowledge, you organized the data using spreadsheets. Spreadsheets are a computer application that is used to organize data. You also learn to use pie charts,bar graphs and other ways to organize data.

SEP11

Git vs Github:

Git and Github is the first unit of SEP11. We learn what Git and Github are and hwo to use then in different situations. Git is a version control system whereas Github is a website where you can see the different versions of your code.

Web Design:

Web design is the second unit of SEP11. In this unit we use the things you've learned in previous years such as javascript and HTML/CSS to design web pages. The layout of a website and how it looks is all apart of web design. You also use git and github to work with others to work on the same website together.

3D Modeling:

3D Modeling is the last unit of SEP11. This unit is a fun unit where you can make literally anything you can imagine. If you have access to a 3D printer you can even print the things you create. The possibilities are endless with this unit.
